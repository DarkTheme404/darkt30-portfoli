# 🥊 TRIUMPH Outreach — инструмент для поиска клиентов

Полная система для поиска заказчиков на разработку сайтов. Состоит из парсеров, базы лидов с контактами, шаблонов писем и email-рассылки.

## 📂 Содержимое

| Файл | Что делает |
|---|---|
| `leads.csv` | **30 готовых лидов** (бизнесы без сайта в Иваново) с соцсетями |
| `generate-leads.js` | Парсер OpenStreetMap — собирает лиды по **любому городу** |
| `find-socials.js` | **Парсер 2ГИС** — ищет бизнесы + достаёт VK, Telegram, Instagram, WhatsApp |
| `send-emails.js` | Email-рассылка через Gmail SMTP с персонализацией |
| `templates.md` | 5 готовых шаблонов + стратегия по каналам |
| `README.md` | Этот файл |

## 🚀 Быстрый старт

### 1. Установка (один раз)
```bash
# Зависимости (один раз)
npm install nodemailer

# Настрой Gmail для рассылки
export SMTP_USER="you@gmail.com"
export SMTP_PASS="xxxx-xxxx-xxxx"   # App Password
export FROM_NAME="Артур Иванов"

# Опционально: 2ГИС API для парсинга соцсетей
# Получи на https://dev.2gis.ru (Catalog API, бесплатно)
export TWO_GIS_API_KEY="rursou44..."
```

### 2. Сгенерируй лидов

**Способ A — Ручной** (30 готовых):
```bash
cat leads.csv
```

**Способ B — Парсер OSM** (без ключа):
```bash
node generate-leads.js "Иваново"
node generate-leads.js "Москва"
node generate-leads.js "Кострома"
```

**Способ C — Парсер 2ГИС** (с соцсетями):
```bash
node find-socials.js "Иваново" "Фитнес-клубы"
node find-socials.js "Москва" "Рестораны"
node find-socials.js "Иваново" "Стоматологии"
```

### 3. Email-рассылка
```bash
node send-emails.js
```

### 4. WhatsApp / Telegram / Звонок
Из `templates.md` возьми шаблоны, подставь имя/категорию, отправь вручную.
