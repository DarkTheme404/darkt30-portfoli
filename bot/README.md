# 🤖 TRIUMPH Outreach Bot

**Локальный Telegram-бот** для управления outreach-кампанией с AI-генератором писем.

## 📱 Команды

### 📋 Лиды
| Команда | Что делает |
|---|---|
| `/leads` | Список лидов (10 на страницу) |
| `/leads 2` | Следующая страница |
| `/show 5` | Полная информация о лиде #5 |
| `/find автосервис` | Поиск по названию/категории/адресу |
| `/has-contacts` | Сколько лидов имеют контакты |

### 📡 Контакты
| Команда | Что делает |
|---|---|
| `/contacts 5` | Все ссылки для связи: телефон, email, ВК, Telegram, Instagram, WhatsApp, YouTube |
| `/call 5` | Готовый скрипт звонка с follow-up контактами |

### 🤖 AI
| Команда | Что делает |
|---|---|
| `/gen 5` | Google Gemini генерирует персонализированное письмо для лида #5 |

### ✏️ Управление
| Команда | Что делает |
|---|---|
| `/add Имя\|Кат\|Тел\|Email\|VK\|TG\|IG\|WA` | Добавить лид |
| `/del 5` | Удалить лид #5 |
| `/leads.csv` | Скачать базу CSV |
| `/stats` | Статистика (отправлено/ответов) |
| `/help` | Эта справка |

## 🚀 Установка (3 минуты)

### 1. Создай бота
- Открой [@BotFather](https://t.me/BotFather) → `/newbot` → скопируй токен

### 2. Получи Gemini API
- Открой [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
- Create API key → скопируй

### 3. Настрой
```bash
cd "/Users/artem/triumph 2026/bot"
cp .env.example .env
nano .env
```

Впиши:
```
TELEGRAM_BOT_TOKEN=123456:ABC-DEF...
GEMINI_API_KEY=AIzaSy...
GEMINI_MODEL=gemini-2.0-flash-exp
```

### 4. Запусти
```bash
./start.sh
```

## 💡 Workflow

1. **Утро**: `node ../outreach/find-socials.js "Иваново" "Фитнес-клубы"` — пополни базу
2. **Днём**: открой бота → `/leads` → смотри новых
3. **`/contacts 5`** — посмотри все способы связи
4. **`/gen 5`** — получи AI-письмо
5. Скопируй и отправь через email / WhatsApp / Telegram
6. **`/call 5`** — получи скрипт для звонка
7. **`/stats`** — смотри результат

## 🔐 Безопасность

- Бот работает **локально** — запускается только когда ты сам включаешь `node index.js`
- Токены хранятся в `.env`
- Когда терминал закрыт — бот не работает
- Токен знаешь только ты

## 🆘 Troubleshooting

**Бот не отвечает:**
- Проверь токен в `.env`
- Проверь, что `node index.js` запущен
- Проверь интернет

**AI не генерирует:**
- Проверь `GEMINI_API_KEY`
- Попробуй сменить модель на `gemini-1.5-flash`

**Ошибка `polling_error`:**
- Перезапусти бота (Ctrl+C, потом `./start.sh`)
